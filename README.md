# Iris Flower Classification

## Project Overview

This project focuses on classifying Iris flowers into three different species: Setosa, Versicolor, and Virginica using machine learning techniques. The model is trained using measurements of sepal length, sepal width, petal length, and petal width.

## Dataset

- Iris Dataset
- Features:
  - Sepal Length
  - Sepal Width
  - Petal Length
  - Petal Width
- Target:
  - Iris Species

## Technologies Used

- Python
- Pandas
- NumPy
- Matplotlib
- Scikit-learn

## Machine Learning Model

- Random Forest Classifier

## Project Workflow

- Data Loading
- Data Exploration
- Data Preprocessing
- Model Training
- Model Evaluation
- Prediction

## Results

The Random Forest Classifier achieved high classification accuracy and successfully classified the three Iris flower species based on their measurements.

## Conclusion

This project demonstrates the complete workflow of a machine learning classification problem. It helped me understand data preprocessing, model training, prediction, and evaluation while showing how machine learning can accurately classify different flower species.
